# 视频拆图 VidPic

将视频转换为可打印的翻页动画书。自动提取视频帧，按 A4 纸排版（每页 8 帧），支持打印、ZIP 打包下载和批量图片导出。

## 功能

- **视频上传** — 支持点击选择或拖拽上传 MP4 / WebM / MOV 等格式
- **帧提取** — 可调节每秒帧数（1~30 fps）、提取时长、帧尺寸
- **帧预览与编辑** — 预览提取的帧，支持多选和删除不需要的帧
- **A4 打印排版** — 自动按 2×4 排列生成 A4 页面，带页码
- **多种导出方式** — 浏览器打印 / ZIP 打包下载 / 图片批量下载
- **移动端适配** — 响应式布局，在手机上也能够正常使用

## 使用

访问浏览器打开vidpic.zeora.top即可。

```bash
python -m http.server 8080
```
